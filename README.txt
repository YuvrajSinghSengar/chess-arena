How to use:

1. Open this folder in VS Code.
2. Create a Firebase project, enable Firestore, and copy your web app config.
3. Paste your config in firebase-config.js (replace YOUR_API_KEY etc.).
4. Use Live Server OR any static server and open index.html.

Main Features:
- Welcome page with typewriter intro.
- Login / Sign Up with full profile (name, username, location, education).
- Dashboard showing rating (starts at 1000).
- Learn section with lessons & completion tracking.
- Play Online: finds opponents near your rating and shows top players.
- Play with Friends: add friends by username and list them.
- Play vs Robot: simple chess board vs basic robot, manual result update which changes rating.
- Rankings: filter by global/country/state/city/institute and see rating-based order.
- Invite Match: send and accept match invitations between users.

Hosting:
- You can host this folder directly on GitHub Pages or Netlify.
